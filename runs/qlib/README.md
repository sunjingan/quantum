# Qlib 通用入口

这里放的是不属于 ETF Loop 主线、但仍然是可直接运行的 Qlib/策略研究入口。

常用脚本：

- `backtest_v2.py`
- `backtest_risk.py`
- `my_strategy.py`
- `poe_pb_roe_strategy.py`
- `trend_serenity_quant_strategy.py`
- `test_env.py`

这些脚本主要用于：

- 验证 Qlib 本地环境
- 试验通用 A 股策略
- 跑基础回测或风控对照
- 检查 Tushare / Qlib 数据链路

如果你在找 ETF Loop 主线，请回到 [`../etf_loop/README.md`](../etf_loop/README.md)。
